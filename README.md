# 七国风云图志

一个以原创同人地图为核心的《权力的游戏 / 冰与火之歌》中文资料网站。

## 第一版功能

- 可拖动、缩放的维斯特洛与厄斯索斯示意地图
- 地点点击介绍：临冬城、君临、长城、龙石岛、高庭、凯岩城、多恩、铁群岛等
- 家族势力筛选：史塔克、兰尼斯特、坦格利安、拜拉席恩、提利尔、马泰尔、葛雷乔伊、艾林、徒利、波顿
- 人物路线：琼恩、艾莉亚、提利昂、丹妮莉丝、詹姆、奈德
- 战争时间线：劳勃起义、五王之战、黑水河之战、红色婚礼、私生子之战、临冬城大战、君临陷落
- 人物、家族、战争、武器快速搜索
- 手机端适配

## 使用

直接打开 `index.html` 即可运行，不需要安装依赖。

## 部署到 GitHub Pages

进入仓库 `Settings → Pages`，选择：

- Source：Deploy from a branch
- Branch：main
- Folder：/ (root)

保存后即可生成公开网址。
