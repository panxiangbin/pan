# 七国风云图志

一个以原创同人地图为核心的《权力的游戏 / 冰与火之歌》中文互动资料网站。

## 在线访问

https://panxiangbin.github.io/pan/

## 当前功能

- 可拖动、缩放的维斯特洛与厄斯索斯原创示意地图
- 地点点击介绍：临冬城、君临、长城、龙石岛、高庭、凯岩城、多恩、铁群岛等
- 家族势力筛选：史塔克、兰尼斯特、坦格利安、拜拉席恩、提利尔、马泰尔、葛雷乔伊、艾林、徒利、波顿
- 人物路线：琼恩、艾莉亚、提利昂、丹妮莉丝、詹姆、奈德
- 战争时间线：劳勃起义、五王之战、黑水河之战、红色婚礼、私生子之战、临冬城大战、君临陷落
- 人物、地点、家族、战争与专题资料快速搜索
- 人物关系网络：20 位核心人物、39 条关键关系，可按亲情、爱情、盟友、矛盾和敌对筛选
- 人物关系与地图地点联动
- 手机端详情抽屉与横向关系图浏览
- 网站图标、安装清单、搜索引擎抓取规则和站点地图

## 使用

直接打开 `index.html` 即可运行，不需要安装依赖或构建工具。

## GitHub Pages 部署

仓库已经使用以下设置：

- Source：Deploy from a branch
- Branch：main
- Folder：/ (root)

提交到 `main` 后，GitHub Pages 会自动更新公开网站。

## 文件结构

- `index.html`：主页面与地图 SVG
- `styles.css`：第一版基础样式
- `data.js`：地点、家族、路线、战争和资料库数据
- `app.js`：地图、搜索、路线和时间线交互
- `enhancements.css`：第二版人物关系图样式
- `enhancements.js`：人物关系数据与交互
- `runtime-fixes.js`：地图与关系图事件兼容修复
- `manifest.webmanifest`：网站安装信息
- `robots.txt` / `sitemap.xml`：搜索引擎支持
